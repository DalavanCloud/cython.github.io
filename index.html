About Cython

Cython is a language that makes writing C extensions for the Python language as easy as Python itself. Cython is based on the well-known Pyrex, but supports more cutting edge functionality and optimizations.

The Cython language is very close to the Python language, but Cython additionally supports calling C functions and declaring C types on variables and class attributes. This allows the compiler to generate very efficient C code from Cython code.

This makes Cython the ideal language for wrapping external C libraries, and for fast C modules that speed up the execution of Python code.

    Download

    Cython is freely available under the open source Apache License.

    The latest release of Cython is 0.11.3 (released 2009-09-26). You can download it as a gzipped tar or as a zip file.

    Cython is also available from the PyPi package repository.

    For Windows, Dave Cournapeau has created 0.11.2 installers for Python 2.5 and Python 2.6.
